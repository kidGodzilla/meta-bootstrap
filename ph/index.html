What happens when I add a site?

When you add a site, tests will automatically be created at various resolutions and will be run in batch with any other tests you have created. If there is a failure, you will be notified via email.

Optionally, you may choose to enable page crawling for the site you've added. This will create tests for all the sub-pages in your site as well, as long as they're on the same domain.
