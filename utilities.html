CSS Utilities

MetaBootstrap Built-in Utility Libraries

MetaBootstrap aims to become a comprehensive framework for building websites, and includes a number of CSS utility libraries to make building Bootstrap-based websites fast, easy, and fun.

Responsive Margins & Padding

Utility classes to add, remove, and modify margins and padding. Optionally, responsive-only conditions can be appended for additional flexibility. 

      An eight step powers of two scale ranging from 0 to 16rem.

      Legend:

      p   = padding
      m   = margin

      a   = all
      h   = horizontal (top & bottom)
      v   = vertical (left & right)
      t   = top
      r   = right
      b   = bottom
      l   = left

      n   = none
      xs  = extra small (0.25rem or 4px)
      s   = small (0.5rem or 8px)
      m   = medium (1rem or 16px)
      l   = large (2rem or 32px)
      x   = extra (4rem or 64px)
      xl  = extra large (8rem or 128px)
      xxl = extra extra large (16rem or 256px)

      -xs = Extra Small Devices (Phones, < 480px wide)
      -sm = Small Devices (Phones, < 768px wide)
      -md = Medium Devices (Tablets, < 992px wide)
      -lg = Large Devices (Desktops & Laptops, Anything larger than 992px wide)

      Examples:

      class="pvl"    -> 2rem (32px) of Padding, vertical (top & bottom)
      class="mtm-xs" -> 1rem (16px) of Margin, top, But only for XS (<480px) devices

Text Utilities

Several utility libraries have been included to facilitate common typography-related use-cases, including responsive text justification & on-the-fly font-weight adjustment.

Responsive Text Justification

Allows text justification to be changed as the window is scaled up or down. Follows intuitive Bootstrap conventions. 

      Window Sizes:

      text-xs    -> Extra Small Devices (Phones, < 480px wide)
      text-sm    -> Small Devices (Phones, < 768px wide)
      text-md    -> Medium Devices (Tablets, < 992px wide)
      text-lg    -> Large Devices (Desktops & Laptops, Anything larger than 992px wide)

      Text Justification:

      -left      -> Left-aligns text
      -center    -> Center-aligns text
      -right     -> Right-aligns text
      -justify   -> Justifies text

      Examples:
      text-xs-center  -> Centers text, but only for XS devices (Phones, < 480px wide)
      text-md-left    -> Left-aligns text, but only for MD devices (tablets, < 992px wide)

Giant Text

Creates gigantic headline text, but only for users with larger displays. 

      <h1 class="giant-text">  -> Creates gigantic H1 text on devices < 992px wide.
      <h2 class="giant-text">  -> Creates gigantic H2 text on devices < 992px wide.
      <h3 class="giant-text">  -> Creates gigantic H3 text on devices < 992px wide.
      <h4 class="giant-text">  -> Creates gigantic H4 text on devices < 992px wide.

Other Text Utilities

      class="small-text"      -> Makes the text small
      class="small-caps"      -> Implements pseudo-smallcaps
      class="strong"          -> Makes the text bold
      class="subdued"         -> Shifts the alpha channel to 50%, creating subdued text.
      class="dark-text-glow"  -> Creates a subtle 20px shadow glow around the text. Useful for overlaying text on images.
      class="white-text-glow" -> Creates a subtle 20px white glow around the text. Useful for overlaying text on images.
