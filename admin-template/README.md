# Bootstrap-Admin-Template

> An Admin Template Based On Twitter Bootstrap 3.x


## Building

#### 2.3.2 Version
required [node.js](http://nodejs.org/) & [bower](http://bower.io/) & [grunt](http://gruntjs.com/getting-started)

```shell
    $ cd yourfoldername
    $ npm install
    $ npm run build
    $ grunt serve
```
