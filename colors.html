Color Utilities

MetaBootstrap Built-in Utility Libraries

MetaBootstrap aims to become a comprehensive framework for building websites, and includes a number of CSS utility libraries to make building Bootstrap-based websites fast, easy, and fun.

Predefined Colors

MetaBootstrap comes with a growing number of pre-defined colors which can be used with any Bootstrap theme. They work similarly to Bootstrap's built-in colors.

These classes are either prefixed with bg- (CSS background-color property), or text- (CSS color property).
